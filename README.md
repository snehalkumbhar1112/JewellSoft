💍 JewellSoft 

JewellSoft is a GST-compliant jewelry retail management system designed specifically for gold and silver businesses. It simplifies daily operations like billing, inventory tracking, and rate management while handling weight-based transactions with precision.

🚀 Key Features :
📊 GST-Compliant Billing
  Accurate invoice generation with tax calculations
  Designed as per Indian GST standards
💰 Real-Time Gold & Silver Rates
  Daily rate updates
  Auto-calculation of product pricing based on current rates
⚖️ Weight-Based Sales Tracking
  Sales handled in grams (gold/silver)
  Precise calculation based on weight and purity
📦 Inventory Management
  Track stock by weight and quantity
  Real-time inventory updates
🏬 Multi-User & Retail Support
  Used by multiple jewelry retailers
  Role-based access for staff and admin
🔄 Stock Logic (Similar to TradeLedger)
  Consistent and scalable inventory handling
  Seamless tracking of inward/outward stock
  
🧩 Core Modules
1. 🛒 Sales Module
  Weight-based product selling (Gold/Silver)
  Auto price calculation using daily rates
  GST-enabled billing

3. 📥 Purchase Module
  Record gold/silver purchases by weight
  Update stock automatically
  Maintain supplier records

5. 📦 Inventory Module
  Track stock in grams
  Maintain purity and category-wise stock
  Real-time updates across operations

7. 💳 Loan / Metal Balance System
  Manage customer loans in gold/silver (e.g., 1gm, 5gm)
  Track payable/receivable metal balance
  Adjust loans with current market rates

🌐 Scalability
  Actively used by multiple jewelry retailers
  Designed for handling high-value and precise transactions
  
💡 Highlights
  Real-time metal rate integration
  Weight-based accounting system
  Loan system in metal units (gold/silver)
  Accurate and secure transaction handling
